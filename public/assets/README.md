## Image Requirements

- Format: WebP
- Recommended size: 512x512px or smaller
- Transparent backgrounds preferred
- Keep file sizes under 100KB for fast loading

## Fallback Behavior

If a webp image is not found, the application will fall back to:
1. Emoji representation
2. Text labels

This allows the app to work even without custom images.
